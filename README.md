# 바디 버디

## commit convention
### convention 형식
``` 
[convention 명] : [#이슈번호] 커밋 내역
[내용]
```

- 제목과 본문은 한 줄 띄기
- 제목 첫 글자는 대문자로 작성
- 제목 끝에 `.` 붙이지 말 것

<br />

### convention 종류
| 태그이름 | 설명                                                  |
| -------- | ----------------------------------------------------- |
| feat     | 새로운 기능 추가                                      |
| fix      | 버그 수정                                             |
| design   | css 등 사용자 UI 수정                                 |
| style    | 코드 포맷 변경, 세미 콜론 누락, 코드 수정이 없는 경우 |
| refactor | 코드 리팩토링                                         |
| comment  | 필요한 주석 추가 및 변경                              |
| docs     | 문서 수정                                             |
| chore    | 패키지 매니저 설정                                    |
| rename   | 파일 혹은 폴더명 수정하거나 옮기는 작업               |
| remove   | 파일을 삭제하는 작업만 하는 경우                      |
